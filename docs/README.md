# examples-js-vue docs


## The way to the present

### clone

```shell
git clone https://github.com/suzu-devworks/examples-js-vue.git
cd examples-js-vue
```

### examples-js-vue-started

```shell
mkdir -p apps/examples-js-vue-started 
cd apps/examples-js-vue-started 
vue create .

# Vue CLI v5.0.6
# ? Generate project in current directory? Yes
# 
# Vue CLI v5.0.6
# ? Please pick a preset: Default ([Vue 3] babel, eslint)
# 
# Vue CLI v5.0.6
# ✨  Creating project in /workspaces/apps/examples-js-vue-started.
# ⚙️  Installing CLI plugins. This might take a while...

vue add router
vue add vuex
```
